# Ejemplo Media Queries
## descripcion
Este es un proyecto ejempplo utilizado para mostrar como podemos utiilzar las media queries en css para poder darle diseño responsivo a nuestras paginas web, y asi se puedan ver en los dispotivos mobiles como por ejemplo en en dispotivos mobiles de vean 2 columnas en ves de 5 que se pueden llegar a ver en monitores grandes de pc

|html |usado para darle estructura al contenido y asi que el navegador lo pueda interpretar|
